# movepub: Prepare Movebank Data for Publication

Prepare animal tracking data from 'Movebank'
(<https://www.movebank.org/>) for publication in a research repository.
With 'movepub' you can document data with metadata following the Data
Package standard and transform these to Darwin Core and Ecological
Metadata Language ('EML') for publication to the Global Biodiversity
Information Facility ('GBIF') and the Ocean Biodiversity Information
System ('OBIS').
